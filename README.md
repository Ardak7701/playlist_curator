# Playlist Curator

## Project Description
Playlist Curator is a Python-based music playlist management system.
The application allows users to:

- Add and remove songs
- Analyze genres
- Find similar songs
- Export playlists
- Save data using JSON and CSV

---

## Features

- Object-Oriented Programming
- Functional Programming
- Regex Validation
- Iterators and Generators
- Custom Decorators
- JSON and CSV File Support
- Unit Testing
- Modular Architecture

---

## Technologies Used

- Python 3
- unittest
- json
- csv
- os
- re
